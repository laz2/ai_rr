Класс операторов удаления erase

 

Содержание главы

1.     Оператор удаления элемента eraseEl

2.     Оператор удаления трёхэлементной конструкции eraseElStr3

3.     Оператор удаления пятиэлементной конструкции eraseElStr5

 

 

Класс операторов удаления предназначен для удаления как одного элемента, так и конструкции языка SCP. Для удаления элемента конструкции необходимо использовать специализированный атрибут (f_).

 

Следует различать удаление элемента из памяти и удаление элемента из множества. Так как во втором случае удаляется только дуга, соединяющая узлы множества и элемента.

 

 

Оператор удаления элемента eraseEl

 

Синтаксис: 

eraseEl([

            1_: element

         ], goto_yes, goto_no, goto_error)

 

Параметры:

element

[in] элемент удаления (уточняется атрибутами, смотрите примеры ниже)

goto_yes

метка успешного выполнения, если элемент удален

goto_no

метка безуспешного выполнения, если элемент не удален

goto_error

метка ошибочного выполнения, если в ходе удаления возникла ошибка

 

Назначение:

Данный оператор предназначен для удаления одиночного элемента. Значение операнда оператора должно быть известно (найдено), так как в противном случае операнд не будет удален.

Обязательным атрибутом является 1_. Все остальные атрибуты являются не обязательными. Порядок следования атрибутов не влияет на результат удаления.

 

Пример 1. Необходимо удалить константный узел.

 

eraseEl([

             1_: fixed_: node_: const_: node

         ])

 

Пример 2. Необходимо удалить позитивную дугу.

 

eraseEl([

             1_: assign_: arc_: pos_: arc_elem

         ])

 

Таким образом одним из способов удаления элемента из множества является удаление дуги соединяющей его со знаком (узлом) множества.

 

 

Оператор удаления трёхэлементной конструкции eraseElStr3

Синтаксис: 

eraseElStr3([

                1_: node,

                2_: arc,

                3_: element

             ], goto_yes, goto_no, goto_error)

 

Параметры:

node

[in] узел, который является первым элементом в трёхэлементной конструкции

arc

[in] дуга, которая является вторым элементом в трёхэлементной конструкции

element

[in] элемент, который является третьим элементом в трёхэлементной конструкции

goto_yes

метка успешного выполнения, если элемент удален

goto_no

метка безуспешного выполнения, если элемент не удален

goto_error

метка ошибочного выполнения, если в ходе удаления возникла ошибка

 

Назначение:

Оператор предназначен для удаления элементов трехэлементной конструкции. Для указания удаляемого элемента применяется атрибут f_. Значения удаляемых операндов должно быть известны (найдены), так как в противном случае операнды не будет удалены.

 

Пример 3. Необходимо удалить элемент node из множества set.

 

eraseElStr3([

                 1_: fixed_: const_: node_: set,

                 2_: assign_: pos_: f_: arc,

                  3_:  fixed_: node

             ])

Для удаления элемента из множества достаточно удалить дугу, соединяющую его с этим множеством. При этом сам элемент (node) не удаляется из памяти.

 

Оператор удаления пятиэлементной конструкции eraseElStr5

Синтаксис: 

eraseElStr5([

                1_: node,

                2_: arc,

                3_: element,

                4_: attr_arc,

                5_: attr_node

             ], goto_yes, goto_no, goto_error)

 

Параметры:

node

[in] узел, который является первым элементом в пятиэлементной конструкции

arc

[in] дуга, которая является вторым элементом в пятиэлементной конструкции

element

[in] элемент, который является третьим элементом в пятиэлементной конструкции

attr_arc

[in] дуга, которая является четвёртым элементом в пятиэлементной конструкции

attr_node

[in] узел, который является пятым элементом в пятиэлементной конструкции

goto_yes

метка успешного выполнения, если конструкция создана

goto_no

метка безуспешного выполнения, если конструкция не создана

goto_error

метка ошибочного выполнения, если в ходе удаления возникла ошибка

 

Назначение:

Оператор предназначен для удаления элементов пятиэлементной конструкции. Для указания удаляемого элемента применяется атрибут f_. Значения удаляемых операндов должно быть известны (найдены), так как в противном случае операнды не будет удалены.

Пример 4. Необходимо удалить (снять) атрибут attr элемента node, входящего в состав множества set

 

eraseElStr5([

                 1_: fixed _: const_: set,

                 2_: fixed _: const_: pos_: arc,

                 3_: fixed _: node_ : elmn,

                 4_: fixed _: const_: pos_: f_: attr_arc,

                 5_: fixed_: const_: attr

             ])

 

 

Содержание главы

4.     Оператор удаления элемента eraseEl

5.     Оператор удаления трёхэлементной конструкции eraseElStr3

6.     Оператор удаления пятиэлементной конструкции eraseElStr5
