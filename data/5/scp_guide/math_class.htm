Класс операторов математических действий

 

Содержание главы

1.    Оператор сложения содержимого add

2.    Оператор вычитания численного содержимого sub

3.    Оператор умножения численного содержимого mult

4.    Оператор деления численного содержимого div

5.    Оператор возведения в степень численного содержимого pow

 

Данный класс операторов предназначен для проведения математических операций над численным содержимым узлов.

 

Оператор сложения содержимого add

 

Синтаксис:

add([

      1_: result,

      2_: node1,

      3_: node2

     ], goto_yes, goto_no, goto_error)

 

Параметры:

result

[out] узел суммы, в содержимое которого заносится результат операции

node1

[in] узел, содержимое которого является первым слагаемым

node2

[in] узел, содержимое которого является вторым слагаемым

goto_yes

метка перехода, если операция успешно завершена (содержимое result сформировано)

goto_no

метка перехода, безуспешное завершение операции

goto_error

метка ошибочного выполнения

 

Назначение:

Оператор осуществляет сложение двух чисел, являющихся содержимым  узлов node1, node2. После успешного выполнения оператора содержимым узла result будет результат суммирования.

При организации циклов необходимы счётчики – переменные, которые последовательно увеличиваются на значение шага. Это позволяет делать оператор add. Но оператор add может использоваться и для  конкатенации (сложения) строкового содержимого узлов node1 и node2.

 

Внимание: Довольно часто необходимо переменную увеличить на какое-то значение, то есть результат нужно занести не в отдельный узел, а в слагаемое. Для этого не нужно использовать промежуточную переменную. Операнд можно помечать атрибутом fixed_, но его значение перед выполнением оператора должно быть найдено. Иначе возникнет ошибка, так как атрибутом fixed_ можно помечать только операнд с найденным значением.

Внимание: При использовании атрибута fixed_ (применительно к первому операнду оператора add) содержимое узла будет дополнено. В случае assign_ содержимое будет сформировано заново.

Пример 1. Сложение двух чисел, которые являются содержимым константных узлов one и two.

 

// В секции констант программы пишем

one =n= 1;

two =n= 2;

...

// Вызываем оператор add

add([1_: assign_: result,

     2_:  fixed_: one,

     3_:  fixed_: two])

В результате выполнения данного оператора будет сформировано числовое содержимое узла result.

Пример 2. Используем переменную i в качестве счетчика. Необходимо увеличить счетчик на единицу.

 

// В секции констант программы пишем

1 =n= 1;

...

// Вызываем оператор add

add([1_: 2_:  fixed_: i,

     3_:  fixed_: 1])

То есть в качестве одного слагаемого выступает результат самого сложения. Следовательно допускается данное сокращение в тексте оператора add.

 

Оператор вычитания численного содержимого sub

 

Синтаксис: 

sub([

      1_: result,

      2_: node1,

      3_: node2

     ], goto_yes, goto_no, goto_error)

 

Параметры:

result

[out] узел разности, в содержимое которого заносится результат операции

node1

[in] узел, содержимое которого уменьшаемое

node2

[in] узел, содержимое которого вычитаемым

goto_yes

метка перехода, если операция успешно завершена (содержимое result сформировано)

goto_no

метка перехода, безуспешное завершение операции

goto_error

метка ошибочного выполнения

Назначение:

Оператор позволяет находить разность двух чисел. После успешного выполнения оператора в содержимое узла result будет помещена разность двух числовых содержимых узлов  node1 и node2.

Внимание: При использовании атрибута fixed_ (применительно к первому операнду оператора sub) содержимое узла будет дополнено (уменьшено). В случае assign_ содержимое будет сформировано заново.

Пример 3. Разность  двух чисел, которые являются содержимым константных узлов one и two. Результат помещаем в содержимое переменной result.

 

// В секции констант программы пишем

one =n= 1;

two =n= 2;

...

// Вызываем оператор add

sub([1_: assign_: result,

     2_:  fixed_: two,

     3_:  fixed_: one])

В результате выполнения данного оператора будет сформировано числовое содержимое узла result. В случае если необходимо вычесть из разности, то справедлива короткая запись оператора (смотрите пример 2 этой главы)

 

Оператор умножения численного содержимого mult

 

Синтаксис: 

mult([

      1_: result,

      2_: node1,

      3_: node2

      ], goto_yes, goto_no, goto_error)

 

Параметры:

result

[out] узел, в содержимое которого заносится результата операции умножения

node1

[in] узел, содержимое которого является первый множитель

node2

[in] узел, содержимое которого является второй множитель

goto_yes

метка перехода, если операция успешно завершена (содержимое result сформировано)

goto_no

метка перехода, безуспешное завершение операции

goto_error

метка ошибочного выполнения

 

Назначение:

Оператор используется для умножения численного содержимого узлов node1 и node2. Результат операции заносится в содержимое узла result. Для данного оператора справедливы все правила и ограничения операторов add и sub.

Оператор деления численного содержимого div

 

Синтаксис: 

div([

      1_: result,

      2_: node1,

      3_: node2

     ], goto_yes, goto_no, goto_error)

 

Параметры:

result

[out] узел частного, в содержимое которого заносится результата операции

node1

[in] узел, содержимое которого является делимое

node2

[in] узел, содержимое которого является делитель

goto_yes

метка перехода, если операция успешно завершена (содержимое result сформировано)

goto_no

метка перехода, безуспешное завершение операции

goto_error

метка ошибочного выполнения

Назначение:

Оператор используется для деления численного содержимого узлов node1 и node2. Результат операции заносится в содержимое узла result. Для данного оператора справедливы все правила и ограничения операторов add и sub, mult.

 

Оператор возведения в степень численного содержимого pow

 

Синтаксис: 

pow([

      1_: result,

      2_: node1,

      3_: node2

     ], goto_yes, goto_no, goto_error)

 

Параметры:

result

[out] узел, в содержимое которого заносится результата операции возведения в степень

node1

[in] узел, содержимое которого является число, которое необходимо возвести в степень

node2

[in] узел, содержимое которого является число – степень

goto_yes

метка перехода, если операция успешно завершена (содержимое result сформировано)

goto_no

метка перехода, безуспешное завершение операции

goto_error

метка ошибочного выполнения

 

Назначение:

Используется для возведения в степень численного содержимого узла node1. Степень записана в содержимом узла node2. Результат операции заносится в содержимое узла result. Для данного оператора справедливы все правила и ограничения операторов add и sub, mult, div.

1.     Оператор сложения содержимого add

2.     Оператор вычитания численного содержимого sub

3.     Оператор умножения численного содержимого mult

4.     Оператор деления численного содержимого div

5.     Оператор возведения в степень численного содержимого pow
