Класс операторов для работы с идентификаторами элементов idtf

 

Содержание главы

1.     Оператор установки идентификатора элемента idtfAssign

2.     Оператор очистки идентификатора элемента idtfErase

3.     Оператор получения идентификатора элемента idtfMove

 

Данный класс операторов предназначен для работы с идентификаторами элементов. Для проверки наличия установленногоидентификатора у элемента необходимо использовать оператор ifFormIdtf из класса условных операторов.

 

Оператор установки идентификатора элемента idtfAssign

 

Синтаксис: 

idtfAssign([

        1_: fixed_: element,

        2_: fixed_: idtf

       ])

 

Параметры:

element

[in] константа или переменная с вычисленным значением, идентификатор которых будет установлен

idtf

[in] константа или переменная с вычисленным значением со строковым содержимым, которое будет использовано для установки идентификатора

Назначение:

Оператор реализует присвание идентификатора элементу.

 

Оператор для очистки идентификатора элемента idtfErase

Синтаксис: 

idtfErase([

            1_: fixed_: element

         ])

 

 

Параметры:

element

[in] константа или переменная с вычисленным значением, идентификатор которых будет очищен

 

Оператор получения идентификатора элемента idtfMove

 

Синтаксис: 

idtfMove([

        1_: fixed_: target,

        2_: fixed_: source

       ])

 

Параметры:

target

[in] константа или переменная с вычисленным значением, содержимое которых будет записан идентификатор

source

[in] константа или переменная с вычисленным значением, идентификатор которых будет записан в target
