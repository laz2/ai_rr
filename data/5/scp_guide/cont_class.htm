Класс операторов для установки содержимого элемента cont

 

Содержание главы

1.     Оператор установки содержимого элемента contAssign

2.     Оператор очистки содержимого элемента contErase

3.     Оператор преобразования содержимого к строковому содержимому to_str

 

Данный класс операторов предназначен для установки и очистки содержимого констант или значений scp-переменных. Для проверки наличия установленного содержимого у элемента необходимо использовать оператор ifFormCont из класса условных операторов.

 

Оператор установки содержимого элемента contAssign

 

Синтаксис: 

contAssign([

        1_: fixed_: element1,

        2_: fixed_: element2

       ])

 

Параметры:

element1

[in] константа или переменная с вычисленным значением, содержимое которых будет установлено

element2

[in] константа или переменная с вычисленным значением, содержимое которых будет записано в element1

Назначение:

Оператор реализует присвание содержимого элементу.

 

Оператор для очистки сожержимого элемента contErase

Синтаксис: 

contErase([

            1_: fixed_: element

         ])

 

 

Параметры:

element

[in] константа или переменная с вычисленным значением, содержимое которых будет очищено

 

Оператор преобразования содержимого к строковому содержимому to_str

 

Синтаксис: 

to_str([

        1_: fixed_: target,

        2_: fixed_: source

       ])

 

Параметры:

target

[in] константа или переменная с вычисленным значением, в содержимое которых будет записан результат преобразования

source

[in] константа или переменная с вычисленным значением, содержимое которых будет преобразовано в строку
