Класс операторов выборки select

 

Содержание главы

1.     Оператор выборки selectYStr3

2.     Оператор выборки selectNStr3

3.     Оператор выборки selectYStr5

4.     Оператор выборки selectNStr5

 

 

Операторы данного класса позволяют обрабатывать два множества элементов, а именно проводить выборку по заданному условию. Операторы, которые в составе своего названия имеют Y (selectYStr3, selectYStr5), позволяют сформировать или дополнить множество из указанных значений операндов, удовлетворяющим заданному условию. Соответственно N, если условие не выполняется. Более подробно каждый из операторов выборки рассмотрим отдельно.

 

 

Оператор выборки selectYStr3

Синтаксис: 

selectYStr3([

                1_: node,

                2_: arc,

                3_: element,

                set1_: resultSet1,

                set2_: resultSet2,

                set3_: resultSet3

             ], goto_yes, goto_no, goto_error)

 

 

Параметры:

node

[in] узел, который является первым элементом в трёхэлементной конструкции. Данная конструкция является шаблоном для поиска  

arc

[in] дуга, которая является вторым элементом в трёхэлементной конструкции

element

[in, out] элемент, который является третьим элементом в трёхэлементной конструкции

resultSet1

[out] множество, в котором остаются первые элементы конструкций, удовлетворяющие шаблону поиска

resultSet2

[out] множество, в котором остаются вторые элементы конструкций, удовлетворяющие шаблону поиска

resultSet3

[out] множество, в котором остаются третьи элементы конструкций, удовлетворяющие шаблону поиска

goto_yes

метка успешного выполнения, если найдены конструкции, удовлетворяющие шаблону поиска

goto_no

метка безуспешного выполнения, если не найдены конструкции, удовлетворяющие шаблону поиска

goto_error

метка ошибочного выполнения, если в ходе выполнения операции возникла ошибка

 

Назначение:

Под шаблоном поиска пониматься трехэлементная конструкция. После успешного выполнения оператора во множестве resultSet (resultSet1, ... , resultSet3) остаются только те элементы, которые удовлетворяют шаблону поиска.

 

Пример 1. Во множестве set_b,  содержащем элементы elem1, elem2, elem3 необходимо оставить только те элементы, которые входят во множество set_a. Множество set_a состоит из elem3, elem4.

 

selectYStr3([

                 1_: fixed _: set_а,

                 2_: assign_: const_: pos_: arc,

                 3_: assign_: node_ : node,

                 set3_: assign_: set_b

              ])

Оператор выборки selectNStr3

 

Синтаксис: 

selectNStr3([

                1_: node,

                2_: arc,

                3_: element,

                set1_: resultSet1,

                set2_: resultSet2,

                set3_: resultSet3

             ], goto_yes, goto_no, goto_error)

 

 

Параметры:

node

[in] узел, который является первым элементом в трёхэлементной конструкции. Данная конструкция является шаблоном для поиска  

arc

[in] дуга, которая является вторым элементом в трёхэлементной конструкции

element

[in, out] элемент, который является третьим элементом в трёхэлементной конструкции

resultSet1

[out] множество, в котором остаются первые элементы конструкций, не удовлетворяющие шаблону поиска

resultSet2

[out] множество, в котором остаются вторые элементы конструкций, не удовлетворяющие шаблону поиска

resultSet3

[out] множество, в котором остаются третьи элементы конструкций, не удовлетворяющие шаблону поиска

goto_yes

метка успешного выполнения, если найдены конструкции, не удовлетворяющие шаблону поиска

goto_no

метка безуспешного выполнения, если не удовлетворяющие шаблону поиска конструкции не найдены

goto_error

метка ошибочного выполнения, если в ходе выполнения операции возникла ошибка

 

Назначение:

Под шаблоном поиска пониматься трехэлементная конструкция. После успешного выполнения оператора во множестве resultSet (resultSet1, ... , resultSet3) остаются только те элементы, которые не удовлетворяют шаблону поиска.

 

Пример 2. Во множестве set_b,  содержащем элементы elem1, elem2, elem3 необходимо оставить только те элементы, которые не входят во множество set_a. Множество set_b состоит из elem3, elem4.

 

selectNStr3([

                 1_: fixed _: set_а,

                 2_: assign_: const_: pos_: arc,

                 3_: assign_: node_ : node,

                 set3_: fixed_: set_b

             ])

resultSet4

[out] множество, в котором остаются четвертые элементы конструкций, удовлетворяющие шаблону поиска

метка безуспешного выполнения, если не найдены конструкции, удовлетворяющие шаблону поиска

метка ошибочного выполнения, если в ходе выполнения операции возникла ошибка
